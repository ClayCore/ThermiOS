# Source files

This directory contains the majority of the source code within ThermiOS

## Vixen library

In short: utility functions and structures which help with building the kernel and userspace code.

In theory it helps me build the the entire project, because it provides a lot of useful stuff, such as base64 encode, url parsing, custom runtime assertions, various memory handlers and anything useful I come across from other languages or from the C++ standard library.
If it appears like I'm reinventing the wheel, it's because I am...
